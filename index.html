Windows 10 Pro

Windows 10 es un sistema operativo de Microsoft, sucesor de Windows 8.1, diseñado para computadoras de escritorio, portátiles y dispositivos móviles. Ofrece una interfaz de usuario adaptable a diferentes tipos de dispositivos y métodos de entrada. Incluye características como el regreso del menú Inicio, un centro de notificaciones, el asistente virtual Cortana y el navegador Microsoft Edge.
